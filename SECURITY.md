# Security Policy

## Supported Versions

The following versions of **FlexiDatePicker** are currently supported with security updates:

| Version | Supported          |
| -------- | ------------------ |
| 1.x.x    | :white_check_mark: |


Only the latest **1.x.x** releases receive active security maintenance.  
Older versions will not receive patches unless a critical vulnerability is discovered.

---
