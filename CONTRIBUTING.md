## ğŸ›  Development Setup

1. **Fork** this repository.
2. **Clone** your fork locally:
   ```bash
   git clone https://github.com/leoanangmh/flexidatepicker.git
   cd flexidatepicker
   ```
3. **Install Dependencies**
   ```
   npm install
   ```
4. **Run Dev Build**
   ```
   npm run dev
   ```
4. **Build for Production**
   ```
   npm run build
   ```
